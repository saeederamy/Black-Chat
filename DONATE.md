## 🌱 Why Donate?

Black Chat is a labor of love. It's:

- ✅ **100% open-source** — no paid tiers, no premium features
- ✅ **Privacy-respecting** — no analytics, no tracking, no ads
- ✅ **Maintained by one developer** in spare time
- ✅ **Free forever** — no subscription, no lock-in

Your donations directly fund:

| What | Why |
|------|-----|
| 🖥️ **Test servers** | I need real VPS instances to test new features (especially calls behind NAT and TURN-TLS) |
| 🌐 **Domain & SSL** | Demo & testing domains for screenshots and CI |
| ⏰ **Development time** | More donations = more hours dedicated to fixing bugs and adding features |
| 📡 **Bandwidth** | Push notification testing, large file upload tests |
| ☕ **Coffee** | The fuel behind late-night coding sessions |

Every contribution, no matter how small, means the world. **Even $5 makes a real difference.**

## 🆓 Free Ways to Support

Don't have crypto? You can still help! These cost nothing but mean a lot:

### ⭐ Star the Repository
Visit [github.com/saeederamy/Black-Chat](https://github.com/saeederamy/Black-Chat) and click the **Star** button. It helps others discover the project and signals to me that people care.

### 🐛 Report Bugs
Found a bug? [Open an issue](https://github.com/saeederamy/Black-Chat/issues/new) with:
- Steps to reproduce
- Browser & OS
- Screenshot if relevant
- Browser console output (F12)

Good bug reports are gold.

### 💡 Suggest Features
Have an idea? Share it in [GitHub Discussions](https://github.com/saeederamy/Black-Chat/discussions) or open a feature-request issue.

### 📢 Spread the Word
- Share Black Chat on social media
- Recommend it to friends who want privacy
- Write a blog post or tutorial
- Make a YouTube video

### 🔧 Contribute Code
Pull requests are welcome! Whether it's:
- Fixing a typo
- Improving documentation
- Adding a feature
- Optimizing performance

Every PR is appreciated.

### 🌍 Translate
Black Chat will need translations as it grows. If you're fluent in a language other than English/Persian and want to help, get in touch via Issues.

## 🤝 How Donations Are Used

Full transparency. Donations help with:

1. **Server costs** (~70%) — VPS for testing, demo deployment, TURN bandwidth
2. **Domain renewals** (~10%) — Annual costs for test domains
3. **Developer time** (~20%) — Real motivation to spend weekends fixing bugs

I don't keep books publicly, but if you ask, I'll share what's been received.

## 🌟 Want to Sponsor a Feature?

If you'd like a specific feature prioritized for your use case, reach out:

- 📧 Open an issue tagged `[sponsor]` describing what you want
- We'll discuss scope and a fair donation amount
- Feature ships open-source for everyone (no exclusivity)

This is a great way for companies or power-users to fund features they need while contributing to the commons.
